### Hi üëã! I present a simple application that you can use to keep track of your study day, in which you can use the different options it offers. OrganizerIT is an app in which you have tools to use as a guest, but if you register, you can use an additional option in which you will find a CRUD to store, read, save or delete notes. It is still in development, any type of idea or suggestion contact me on my networks .. Enjoy the App!
